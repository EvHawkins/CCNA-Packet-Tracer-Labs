# CCNA Packet Tracer Labs

This repository contains hands-on networking labs built using Cisco Packet Tracer.

## Skills Demonstrated

- VLAN Configuration
- Inter-VLAN Routing
- OSPF
- NAT
- ACLs
- DHCP
- Subnetting
- Network Troubleshooting

## Tools Used

- Cisco Packet Tracer
- CLI Configuration
- TCP/IP Networking
